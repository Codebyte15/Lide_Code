LIDE v3.0.0 Stable Features

  • Chunk Syntax Highlighter: Highlight code efficiently in chunks.
  • In-Built Terminal: Integrated terminal (Ctrl + ,) for faster workflows.
  • Bug Fixes & Responsive UI: Improved stability and interface responsiveness.
  • Multifile Management: Work with multiple files in a single session.
  • Update System Improvement: Smoother and more reliable updates.
  • Select and Replace Keywords: Quickly find and replace text.
  • Enhanced Widget Control: Customize editor behavior via Settings > Preferences.
  • Cache and Saved File Management: Delete unnecessary files safely.
  • Static HTML Data Fetch: Import and read data from static websites.
  • Better File Management: Organize your projects efficiently.
