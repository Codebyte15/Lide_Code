Features

  • Basic Code Editor For Indie Developers
  • Integrated terminal
  • Multi-file management
  • Auto Updater for new versions
  • Find and Replace Text Feature
  • Enhanced preferences Tabs for Customization
  • Login Info Viewer
  • Static HTML Source Code Fetcher Options
  • Preview Pane On right side
  • Explorer On Left Side
  • Plugin System with great control
  • Chunk-based syntax highlighting
  • Auto Indentation
  • Auto Saver Option
  • Very Fast Startup
  • Less RAM and Space required to run

Upcoming Updates & Ongoing Development ‚Äì LIDE v3.1.0

  • The Live Server feature has been removed due to Windows Firewall and permission-related limitations.
  • Security enhancements are currently being tested to ensure a safer and more stable release.
  • Active feedback improvements are in progress to enhance user experience and responsiveness.
  • A secure Version Checker, Downloader, and Installer system is under development.
  • LIDE will now be installed in the AppData directory to improve system compatibility and permission handling.
  • Previous session tabs will be automatically restored on startup.
  • Explorer-related issues have been resolved, including synchronized scrollbars for improved usability.
  • New icon-based action buttons have been added for faster workflow and improved accessibility.
  • Multiple bug fixes, performance optimizations, and speed improvements have been implemented.
  • Enhanced plugin management and faster widget updates for improved extensibility.
  • Core files have been reorganized and modularized to ensure faster startup and better maintainability.

Note:

The features listed above are currently under development and may be modified, postponed, or removed in future updates.
