Portable Builds

Portable builds are distributed as archive packages without an installer. After extraction, LIDE can be executed directly from the extracted directory. These builds do not write to the system registry or modify system-level settings.

LIDE v3.0.0 Features

  • Chunk-based syntax highlighting
  • Integrated terminal
  • Multi-file management
  • Improved update system
  • Find and replace
  • Enhanced preferences
  • Cache cleanup tools
  • Static HTML data fetch
  • Improved file explorer
